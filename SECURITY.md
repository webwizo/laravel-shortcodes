# Security Policy

## Supported Versions

The following versions are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately:

- Open a private GitHub Security Advisory for this repository (preferred).
- If that is not available, contact the maintainers privately and include the details below.

Please include:

- A clear description of the issue and potential impact.
- Steps to reproduce (proof-of-concept if possible).
- Affected versions and environment details.
- Any suggested mitigation or fix.

What to expect:

- Initial acknowledgment within 3 business days.
- Regular updates as the report is triaged and validated.
- If confirmed, a fix will be prepared and released, and a coordinated disclosure timeline will be shared.

Please do not disclose security issues publicly until a fix is available.
